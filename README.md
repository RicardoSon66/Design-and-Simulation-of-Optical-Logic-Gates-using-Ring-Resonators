#**Introduction**
이번 프로젝트는 링 공진기를 이용한 기본적인 게이트 구현(AND,OR,NOT) 및 시뮬레이션 입니다.(추후 다른 게이트도 업데이트 할 예정)  
기본적으로 NOT Gate를 제외하고 입력이 두가지 이므로 프로젝트에 채용한 소자는 Dual-port Ring Resonator 입니다.  
처음에 생각하였던 구조는 트랜지스터로 논리게이트를 구현하는것과 같이 Resonator를 트랜지스터 처럼 활용하면 되겠다고 생각을 하였지만  
이번 프로젝트를 진행하면서 완전히 다르게 설계가 되는것을 확인하였고 꽤 흥미로운 현상을 볼 수 있었습니다.(사실 전 프로젝트의 연장선 같이 되어버렸지만..)  

# **Optical AND Gate**
먼저 시도를 해본것은 AND Gate 입니다.  
생각보다 구현에 대한 해답이 이 Dual-port Ring Resonator을 건드려 보면서 굉장히 빠른 시간에 찾을 수 있었습니다.  
어떻게 보면 해석의 차이로 AND Gate가 아니라고 생각이 들 수도 있지만 적어도 제가 실험해본 결과를 보았을때  
Dual-port Ring Resonator은 이미 AND Gate 그 자체라는걸 알 수 있습니다.  
먼저 도면부터 보겠습니다.
(도면의 사진)
저는 사진에 표기한것 처럼 해석을 진행 하도록 정하였습니다.
먼저 상단의 도파로 A에 Pulse input을 가해보도록 하겠습니다.
(Pulse input A)  
사진에 나오는 것 처럼 A에서의 빛이 커플링으로 인해 링으로 에너지가 빠져나갑니다. 방향은 그대로 왼쪽에서 오른쪽으로 이동을 합니다.  
그리고 최종으로는 하단의 도파로 B로 에너지가 빠지게 되는데 이때 저희는 출력단의 정의를 C와 D로 하였기에 해당 입력의 출력은 0 입니다.  
이번엔 하단의 도파로 B에 Pulse input을 가해보도록 하겠습니다.
(Pulse input B)
첫번째 사진을 상하 반전한것과 같은 결과를 볼 수 있었습니다.  
B에서 커플링으로 옮겨진 빛이 링을 돌면서 상단의 도파로 A에 옮겨지는것을 볼 수 있었고 A는 입력단 이기 때문에 출력은 0 이라는것을 알 수 있었습니다.  
자 그럼 이제 A와 B에 둘다 Pulse input을 가해보도록 하겠습니다.
(Pulse input A, B)
예상을 한것과 같이 출력이 1이 되는것을 볼 수 있었습니다.
하지만 이때 두가지 관찰점이 있습니다.    
**1.링 내부 빛의 충돌**
각각 입력 A와 B에서 커플링이 일어난 빛의 이동 경로는 A는 시계 방향 B는 반시계 방향 입니다.
이 두 빛은 링 내부에서 3시 방향에 충돌이 일어나는데 이는 보강, 상쇄, 보강, 상쇄간섭이 연속적으로 이루어 지는 것 입니다.  
마치 맥박 현상 처럼 보일 수 있으나 이는 완전히 고정된 정상적인 파형입니다.  
왜냐하면 두 빛의 주파수는 완전 동일하게 주었기 때문입니다.  
또한 두 빛이 충돌을 하고 보강, 상쇄간섭이 연속적으로 이루어질때 빛은 제자리에 고정이 되어 있는것을 볼 수 있습니다.  
그리고 이 정상파가 에너지를 역방향으로 전파를 하는것을 볼 수 있었고 이때 두 빛의 충돌 지점은 9시 방향 입니다.  
이때도 마찬가지로 빛이 보강, 상쇄 간섭이 연속적으로 이루어지지만 빛은 일정시간동안 움직이지를 않습니다.
**2.진정한 맥박 현상?**
진정한 맥박 현상을 관찰하기 위해 입력 B에 다른 주파수를 주고 관찰을 해보도록 하겠습니다.
(사진)
사실 다른게 안보입니다. 이는 입력이 Pulse 이기 때문입니다.  
Pulse의 경우는 충돌과 탈출의 시간이 매우매우 짧기 때문에 별 차이를 볼 수 없었습니다.  
그러면 지속광(Continuous Wave, CW)의 경우는 잘 나오지 않을까? 라는 생각이 들었으므로 지속광으로도 해보았습니다. 결과는 다음과 같습니다.
(사진)
신뢰성을 위해 주파수가 같은 지속광을 입력을 해보고 시뮬레이션을 돌려보았습니다.
(사진)
사진을 비교 하였을때 완전히 다르다는것이 확인이 가능하였습니다.

# **Future Work**
저의 보고서를 읽으시는 분들중 AND Gate 부분을 보았을때 이미 눈치를 채셨을수도 있습니다.  
문제점이 한가지 보이는데 바로 역박향 출력에 대한 부분이 있습니다.  
이를 해결하기 위해서는 소자에 대한 터치 보다는 글을 작성하는 시점에서는 제어공학적 기술을 활용을 해야 할 것 같다는 생각이 듭니다.  
저는 아직 광학소자에 대해 또는 광학 컴퓨팅 시스템에 대해 지식이 모자르고 공부 중 이기 때문에 이미 학계에서는 해답이 나와 있을 수도 있다고 생각합니다.
만약 공부중에 해답을 찾았거나 하면 구현 정도로만 끝내겠지만 아직은 모르는 상태이므로 나중에 해야할 작업으로 남겨 놓겠습니다.
